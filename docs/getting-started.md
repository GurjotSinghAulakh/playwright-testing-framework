# Getting Started

## Prerequisites
- Node.js 20+
- pnpm (via Corepack)

## Install

```bash
corepack enable
pnpm install
```

## Run Tests

```bash
pnpm test:api
pnpm test:e2e
```

## Reports
- HTML reports are written to each app's `playwright-report` folder.
- Raw results are written to each app's `test-results` folder.
